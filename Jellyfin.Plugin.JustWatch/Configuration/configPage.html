JustWatch

Adds a JustWatch id and deep-link to your items. The id can be entered manually per item, or auto-resolved by the opt-in task below.

Lets the scheduled task query JustWatch's unofficial GraphQL API to fill in the JustWatch id for items that lack one. Off by default; the endpoint is undocumented and may change without notice.
